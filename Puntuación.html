📚 Glosario de Funciones

.replace(viejo, nuevo)
Reemplaza todas las ocurrencias de una subcadena por otra. En este ejemplo, se usa para eliminar comas de la frase (aunque la frase original no contiene comas).
texto = "hola,mundo"
resultado = texto.replace(",", " ")
# Resultado: "hola mundo"
.lower()
Convierte todos los caracteres de la cadena a minúsculas. Útil para normalizar texto y hacer comparaciones sin distinción entre mayúsculas y minúsculas.
texto = "PYTHON Es Genial"
resultado = texto.lower()
# Resultado: "python es genial"
.split()
Divide una cadena en una lista de palabras (tokens). Por defecto, divide usando espacios en blanco como separador. En este código, convierte la frase en una lista de palabras individuales.
frase = "python es poderoso"
tokens = frase.split()
# Resultado: ['python', 'es', 'poderoso']
print()
Función integrada de Python que muestra el valor de una variable o expresión en la consola. En este caso, imprime la frase ya limpia y en minúsculas.
print("Hola Python")
# Muestra en consola: Hola Python
