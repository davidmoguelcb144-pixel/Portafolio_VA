📚 Glosario de Funciones y Propiedades

cv2.imread()

Lee y carga una imagen desde un archivo en el disco. La imagen se almacena como una matriz NumPy tridimensional en formato BGR (Blue, Green, Red). Esta función es el punto de partida para cualquier operación de procesamiento de imágenes.

Técnica Base

img.shape

Propiedad de NumPy que retorna una tupla con las dimensiones de la matriz de la imagen. Para imágenes a color devuelve (alto, ancho, canales), mientras que para imágenes en escala de grises devuelve solo (alto, ancho). Es fundamental para conocer el tamaño de la imagen.

Atributo Principal

img.shape[0] - Alto

Accede al primer elemento de la tupla shape, que representa el alto de la imagen o número de filas. Este valor indica cuántos píxeles tiene la imagen verticalmente. Es equivalente al número total de líneas horizontales que componen la imagen.

Dimensión Vertical

img.shape[1] - Ancho

Accede al segundo elemento de la tupla shape, representando el ancho de la imagen o número de columnas. Este valor indica cuántos píxeles tiene la imagen horizontalmente. Es el número de píxeles en cada línea horizontal de la imagen.

Dimensión Horizontal

img.shape[2] - Canales

Accede al tercer elemento de shape, que indica el número de canales de color. Para imágenes BGR estándar este valor es 3 (Azul, Verde, Rojo). Para imágenes en escala de grises, este índice no existe ya que solo tienen un canal de intensidad.

Profundidad de Color

print()

Función integrada de Python que imprime valores en la consola. Permite visualizar información textual como las dimensiones de la imagen. Es esencial para debugging y para comprender las propiedades de las estructuras de datos con las que trabajamos.
