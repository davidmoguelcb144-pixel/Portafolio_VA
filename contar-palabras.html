C√≥digo del Ejemplo

frase = "python es divertido y poderoso"

frase_limpio = frase.replace(",", "").replace(".", "").lower()

tokens = frase_limpio.split()

frecuencia = {}
for palabra in tokens:
    if palabra in frecuencia:
        frecuencia[palabra] += 1
    else:
        frecuencia[palabra] = 1

print(frecuencia)

Glosario de Funciones

str.replace(old, new)
Reemplaza todas las ocurrencias de una subcadena por otra. Se usa encadenadamente para eliminar signos de puntuaci√≥n del texto antes de analizarlo, asegurando que las palabras se procesen correctamente sin caracteres especiales.
str.lower()
Convierte todos los caracteres del texto a min√∫sculas. Es esencial para el an√°lisis de frecuencias ya que asegura que palabras como "Python" y "python" se consideren la misma palabra.
str.split()
Divide la cadena de texto en una lista de palabras (tokens) usando espacios en blanco como separadores. Este proceso de tokenizaci√≥n es fundamental para poder contar cada palabra individualmente.
dict = {}
Crea un diccionario vac√≠o. En Python, los diccionarios son estructuras de datos que almacenan pares clave-valor. Aqu√≠ se usa para almacenar palabras como claves y sus frecuencias como valores.
for (bucle)
Estructura de control que itera sobre cada elemento de una secuencia. En este caso, recorre todos los tokens (palabras) para contar cu√°ntas veces aparece cada una en el texto.
if-else (condicional)
Estructura de control que ejecuta diferentes bloques de c√≥digo seg√∫n una condici√≥n. Aqu√≠ verifica si una palabra ya existe en el diccionario: si existe, incrementa su contador; si no, la agrega con valor inicial de 1.
in (operador)
Operador de pertenencia que verifica si una clave existe en un diccionario. Se utiliza para determinar si la palabra ya ha sido contada anteriormente en el diccionario de frecuencias.
+= (operador de incremento)
Operador de asignaci√≥n aumentada que incrementa el valor de una variable. En este c√≥digo, suma 1 al contador de frecuencia cada vez que una palabra repetida es encontrada.
print()
Funci√≥n que muestra informaci√≥n en la consola. Imprime el diccionario completo con todas las palabras y sus respectivas frecuencias de aparici√≥n en el texto.
