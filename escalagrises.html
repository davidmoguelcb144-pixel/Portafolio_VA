⚫ Escala de Grises

Visión Artificial - Tema 02

💻 Código de Ejemplo

import cv2 img = cv2.imread('equipo.jpg') #devuelve una matriz NumPy que representa la imagen en formato BGR (Azul, Verde, Rojo). gris = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY) #Convierte la imagen de color (BGR) a escala de grises. cv2.imshow('Original', img) #Muestra una ventana con el nombre "Original" que contiene la imagen a color. cv2.imshow('Escala de Grises', gris) #Muestra otra ventana con el título "Escala de Grises" cv2.waitKey(0) #El parámetro 0 significa "esperar indefinidamente". #Si pusieras un número, por ejemplo 1000, esperaría 1000 milisegundos (1 segundo) antes de cerrar las ventanas automáticamente. cv2.destroyAllWindows()

📚 Glosario de Funciones

cv2.imread()

Lee una imagen desde un archivo y la carga en memoria como una matriz NumPy. La imagen se carga en formato BGR (Blue, Green, Red), que es el formato predeterminado de OpenCV. Este formato es diferente del RGB convencional.

cv2.cvtColor()

Convierte una imagen de un espacio de color a otro. En este caso, transforma la imagen desde BGR (color) a escala de grises (GRAY). Esta conversión reduce la información de 3 canales (RGB) a un solo canal de intensidad, simplificando el procesamiento.

cv2.COLOR_BGR2GRAY

Es una constante de OpenCV que indica el tipo de conversión de espacio de color a realizar. Específicamente, convierte de BGR (color) a escala de grises. OpenCV usa BGR en lugar de RGB por razones históricas relacionadas con el hardware de captura de video.

cv2.imshow()

Crea y muestra una ventana con la imagen especificada. El primer parámetro es el título de la ventana y el segundo es la imagen a mostrar. Permite visualizar tanto imágenes a color como en escala de grises simultáneamente en ventanas diferentes.

cv2.waitKey()

Espera durante un tiempo determinado (en milisegundos) a que el usuario presione una tecla. Si el parámetro es 0, espera indefinidamente. Con valores como 1000, esperaría 1 segundo antes de continuar. Es esencial para mantener las ventanas abiertas y capturar eventos del teclado.

cv2.destroyAllWindows()

Cierra todas las ventanas creadas por OpenCV y libera los recursos del sistema asociados. Es importante llamar esta función al finalizar para evitar que las ventanas queden abiertas en memoria consumiendo recursos innecesariamente.
